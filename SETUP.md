# 快速启动指南

## 前置要求

确保您的系统已安装：
- Node.js (版本 18 或更高)
- npm 或 yarn

## 安装步骤

### 1. 安装依赖包

在项目根目录下运行：

```bash
npm install
```

这将安装以下主要依赖：
- Next.js 14
- React 18
- TypeScript
- TailwindCSS
- Lucide React (图标库)

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问：http://localhost:3000

## 项目特性

✅ **已完成功能**
- 🏠 首页概览和统计
- 📚 顺序练习（10道示例题目）
- 🎯 模拟考试（随机抽题，计时功能）
- ❌ 错题本（自动收集错题）
- 📊 历史成绩（考试记录和统计）
- 📱 响应式设计（支持手机、平板、桌面）

## 使用说明

### 顺序练习
1. 点击"顺序练习"开始练习
2. 选择答案后点击"提交答案"查看结果
3. 系统自动记录正确率和错题

### 模拟考试
1. 点击"模拟考试"进入考试页面
2. 阅读考试说明，点击"开始考试"
3. 在规定时间内完成所有题目
4. 交卷后查看详细成绩报告

### 错题本
1. 练习中答错的题目会自动加入错题本
2. 在错题本中重新练习错题
3. 答对后题目会自动从错题本移除

### 历史成绩
1. 查看所有模拟考试记录
2. 点击任意记录查看详细答题情况
3. 查看平均分、最高分等统计信息

## 数据说明

- 题库包含10道示例题目（多选题、单选题、判断题）
- 所有数据存储在浏览器本地（LocalStorage）
- 清除浏览器数据会重置所有记录

## 技术特点

- **现代化技术栈**：使用最新的Next.js 14和React 18
- **类型安全**：完整的TypeScript支持
- **美观界面**：基于TailwindCSS的现代化UI设计
- **响应式布局**：适配各种设备屏幕
- **本地存储**：无需服务器，数据安全可靠

## 故障排除

### 端口被占用
如果3000端口被占用，Next.js会自动使用下一个可用端口（如3001）

### 依赖安装失败
尝试清除缓存后重新安装：
```bash
npm cache clean --force
npm install
```

### 页面无法加载
检查控制台是否有错误信息，确保所有依赖正确安装

## 扩展说明

要添加更多题目，请编辑 `data/questions.json` 文件，按照现有格式添加新的题目对象。

项目已配置完整的开发环境，可以直接进行二次开发和功能扩展。
