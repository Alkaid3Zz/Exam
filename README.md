# 基本级执法资格考试练习系统

基于Next.js开发的在线练习与模拟考试平台，帮助用户顺利通过执法资格考试。

## 功能特性

### 📚 顺序练习
- 按题库顺序逐题作答
- 立即显示正确答案和解析
- 实时统计正确率和进度
- 自动记录答题情况

### 🎯 模拟考试
- 随机抽取题目，模拟真实考试环境
- 2小时倒计时功能
- 交卷后自动评分
- 详细的成绩报告和错题解析

### ❌ 错题本
- 自动收集答错的题目
- 支持重新练习错题
- 答对后自动从错题本移除
- 可手动管理错题列表

### 📊 历史成绩
- 记录每次模拟考试成绩
- 统计平均分、最高分、及格率
- 查看详细答题情况
- 成绩趋势分析

### 📤 题库导入
- 支持Word文档格式导入
- 智能解析题目内容和格式
- 自动识别多选题、单选题、判断题、不定项选择题
- 批量转换为JSON格式
- 实时进度显示和状态反馈
- 直接同步到题库文件
- 支持添加和替换两种模式

### 🗄️ 题库管理
- 题库版本管理和变化检测
- 自动调整练习进度适应题库更新
- 用户数据导入导出
- 练习进度重置功能
- 题库统计信息查看

## 技术栈

- **前端框架**: Next.js 14
- **开发语言**: TypeScript
- **样式框架**: TailwindCSS
- **图标库**: Lucide React
- **数据存储**: LocalStorage

## 题库说明

题库包含基本级执法资格考试的各类题型：

- **多选题**: 可选择多个答案
- **单选题**: 只能选择一个答案  
- **判断题**: 正确或错误

每道题目包含：
- 题目内容
- 选项列表
- 正确答案

## 数据存储

应用使用浏览器的LocalStorage存储用户数据：

- `userProgress`: 练习进度和错题记录
- `examHistory`: 模拟考试历史记录

## 使用说明

### 顺序练习
1. 点击"顺序练习"开始练习
2. 选择答案后点击"提交答案"查看结果
3. 系统自动记录正确率和错题

### 模拟考试
1. 点击"模拟考试"进入考试页面
2. 阅读考试说明，点击"开始考试"
3. 在规定时间内完成所有题目
4. 交卷后查看详细成绩报告

### 错题本
1. 练习中答错的题目会自动加入错题本
2. 在错题本中重新练习错题
3. 答对后题目会自动从错题本移除

### 历史成绩
1. 查看所有模拟考试记录
2. 点击任意记录查看详细答题情况
3. 查看平均分、最高分等统计信息

### 题库导入
1. 点击"导入题库"进入导入页面
2. 选择Word文档文件或复制粘贴内容
3. 系统自动解析题目格式
4. 预览解析结果
5. 选择"同步到题库"（添加模式）或"替换题库"（替换模式）
6. 系统自动更新题库文件并调整练习进度

### 题库管理
1. 点击"题库管理"查看题库状态
2. 查看题库统计信息和题型分布
3. 导出用户数据进行备份
4. 导入之前的用户数据
5. 重置所有练习进度
6. 检查题库版本和状态

支持的Word文档格式示例：
```
一、多选题
1. 题目内容（     ）
A.选项1
B.选项2
C.选项3
D.选项4
正确答案：ABC

二、单选题
1. 题目内容（     ）
A.选项1
B.选项2
正确答案：A

三、判断题
1. 题目内容
正确答案：A
```

## 开发说明

### 添加新题目

在 `data/questions.json` 文件中添加新的题目对象：

```json
{
  "id": 11,
  "type": "single",
  "question": "题目内容",
  "options": [
    "A.选项1",
    "B.选项2",
    "C.选项3",
    "D.选项4"
  ],
  "answer": ["A"]
}
```

### 自定义样式

主要样式类定义在 `app/globals.css` 中：
- `.btn-primary`: 主要按钮样式
- `.btn-secondary`: 次要按钮样式
- `.card`: 卡片容器样式
